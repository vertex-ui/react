## Quick Start

```tsx
import { ThemeProvider, Button } from '@vtx-ui/react';
import '@vtx-ui/react/styles.css';

function App() {
  return (
    <ThemeProvider>
      <Button variant="primary">Click me</Button>
    </ThemeProvider>
  );
}
```

## Components

Browse the sidebar to explore all available components:

- **Button** - Interactive buttons with multiple variants
- **Input** - Text input fields with validation
- **Select** - Dropdown selection controls
- **Table** - Data tables with custom rendering
- **Modal** - Accessible dialog overlays
- **Tooltip** - Contextual information on hover
- **Avatar** - User profile images
- **Badge** - Status indicators and labels

## Design Tokens

The library uses a comprehensive design token system. Check out the **Design Tokens** section to see all available tokens for colors, spacing, typography, and more.

## Accessibility

All components are built with accessibility in mind:

- Semantic HTML
- ARIA attributes
- Keyboard navigation
- Focus management
- Screen reader support

Each component includes accessibility tests using jest-axe to ensure compliance.
