Toast Variants Test

Available Variants:

  • success - Green styling with checkmark icon
  • error - Red styling with X icon
  • warning - Orange styling with warning icon
  • info - Blue styling with info icon
  • default - Gray styling with info icon

Imperative API Usage:

toast.success('Success message');
toast.error('Error message');  
toast.warning('Warning message');
toast.info('Info message');
toast.show('Default message'); // or without variant

Hook API Usage:

const { addToast } = useToast();
addToast('Success message', { variant: 'success' });
addToast('Error message', { variant: 'error' });
addToast('Warning message', { variant: 'warning' });
addToast('Info message', { variant: 'info' });
addToast('Default message'); // defaults to 'default'
